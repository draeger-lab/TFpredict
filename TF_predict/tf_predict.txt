tf_predict
        - Classify a unknown sequences based upon IPRscan results.
        - Features: TF/NonTF classification, Bindingdomain(s) prediction, IPR-Domains to Transfac conversion, Superclass classification
        - five operation modes: TF/NonTF training, superclass training, prediction (console), prediction (galaxy), prediction (galaxybatch)

Input (prediction):

        -predict IPRscan_infile
        -i outfile_ipr.libsvm.model             modelfile for TF/NonTF classification
        -s outfile_ipr_super.libsvm.model       modelfile for superclass classification
        -p path/to/iprscan
        -r outfile_ipr.iprs                     iprfile for TF/NonTF classification
        -sr outfile_ipr_super.iprs              iprfile for Superclass classification
        -g DNA.go                               relevant go-terms (ex. DNA-binding)
        -t transHMan                            hashmap for domain to transfac mapping

Additional:
Input (prediction galaxy):

        -galaxy sequence
        -galaxybatch sequences  sequences in IPRscan compatible format
        -b basedir              basedirectory
        -o outfile              HTML-formatted outfile

Output (prediction galaxy):

        outfile                 prediction results in HTML-format

---

Input (training):

        -train OR -super        training mode (TF/NonTF or Superclass)
        -i infile.libsvm        ipr-vectors in libsvm-format
        -f 5                    number of folds

Outfile (training):

        infile.libsvm.model     model file
        infile.libsvm.roc       txt file with ROC values
